Sipros Data Analysis Workflow¶

  • Uses The Following Objects:
    • SiprosDatabase: for loading and summarizing a protein database
    • SiprosSample: for loading each proteomics sample data
    • SiprosExperiment: for suumarizing multiple samples

Step 1: Load Database¶

  • We'll use matplotlib inline for any plots
  • The SiprosDatabase object provides annotations of protein sequences
    • Taxonomic annotation based on BLASTp best hits
      • currently uses a parsed taxonomy file: protein ID - domain phyla class order family genus species
      • this may be changed in the future to allow import of a BLAST output or other sequence alignment such as kraken, diamond, or bowtie in order to assign cds or contig taxonomy
    • Functional annotation based on hmmscan of bactNOG database.
      • like the taxonomy annotation this uses a parsed file but may be expanded in the future to other types of annotation such as GO terms, SEED, KEGG ontology, or Pfams
  • These annotations will be used to classify and count spectra, protein IDs, peptides, and enrichments
